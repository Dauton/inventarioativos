Como devo cadastrar um ativo?

1º - Tente sempre preencher com o máximo de informações possíveis "máximo de campos preenchidos". Isso facilitará encontrar ele no futuro.

2º - Para DESKTOP e NOTEBOOK no campo hostname, assim como na primeira configuração do equipamento, siga o seguinte padrão:
Para DESKTOP: D-UNIDADE-SN
Para NOTEBOOOK: N-UNIDADE-SN.

3º - Os campos que contém o asterisco * são de preenchimento obrigatório.

4º - Para exportar o inventário de sua unidade, clique no botão com o logo do Excel no canto inferior direito.
O mesmo serve também para importar.

Para rever essas informações, precione o botão de interrogação no canto inferior direito.
